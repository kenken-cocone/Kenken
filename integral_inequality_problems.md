# 積分不等式：面積評価を用いた問題集

## 第1ページ　解説：面積評価による積分不等式の証明

### 面積評価とは

面積評価は、関数のグラフで囲まれた領域の面積を不等号で関係付けることで、積分不等式を証明する手法です。主に以下の2つの戦略があります。

#### 戦略1：曲線の上下関係による評価

関数 $f(x)$ と $g(x)$ について $f(x) \leq g(x)$ が区間 $[a,b]$ で成り立つとき、
$$\int_a^b f(x)dx \leq \int_a^b g(x)dx$$
が成立します。

**例**：$0 \leq x \leq 1$ で $x^2 \leq x$ であるから、
$$\int_0^1 x^2 dx \leq \int_0^1 x dx$$

#### 戦略2：折れ線や直線による近似

曲線を上（下）から直線や折れ線で覆うことで、複雑な積分を評価します。

**例**：凸関数 $f$ について、$x$ での接線を $y = f'(x)(t-x) + f(x)$ とすると、凸性から
$$f(t) \geq f'(x)(t-x) + f(x)$$
が任意の $x, t$ に対して成立します。

#### 戦略3：リーマン和との関係

定積分をリーマン和で近似し、分割領域の面積として評価する方法も重要です。

### 実践例

**問題**：$\displaystyle \int_0^{\pi/2} \sin x \, dx$ の値を面積評価で評価してください。

**解答**：
- 下からの評価：$x \in [0, \pi/2]$ で $\sin x \geq \frac{2x}{\pi}$ が成立します。
  $$\int_0^{\pi/2} \sin x \, dx \geq \int_0^{\pi/2} \frac{2x}{\pi} dx = \frac{2}{\pi} \cdot \frac{(\pi/2)^2}{2} = \frac{\pi}{4}$$

- 上からの評価：$x \in [0, \pi/2]$ で $\sin x \leq 1$ が成立するので、
  $$\int_0^{\pi/2} \sin x \, dx \leq \int_0^{\pi/2} 1 \, dx = \frac{\pi}{2}$$

したがって、$\displaystyle \frac{\pi}{4} \leq \int_0^{\pi/2} \sin x \, dx \leq \frac{\pi}{2}$

## 問題1：基本的な不等式（下からの評価）

$\displaystyle \int_0^1 \sqrt{x} \, dx$ について、以下を示してください。

$$\int_0^1 \sqrt{x} \, dx \geq \frac{2}{5}$$

**ヒント**：$x \in [0,1]$ で適切な下界関数を見つけてください。

## 問題2：指数関数の評価

$\displaystyle \int_0^1 e^x \, dx$ について、以下を証明してください。

$$1 < \int_0^1 e^x \, dx < e - 1$$

**ヒント**：上下から不等号を作るには、$e^x$ の値を特定の点で評価してください。

---

## 問題3：三角関数の複合評価

以下の不等式を証明してください。

$$\frac{2}{\pi} < \int_0^1 \sin(\pi x) \, dx < 1$$

**ヒント**：$\sin(\pi x)$ を直線や簡単な関数で上下から押さえてください。

---

## 問題4：対数関数の評価

以下の不等式を証明してください。

$$\int_1^e \ln x \, dx > 1$$

**ヒント**：$x \in [1, e]$ で $\ln x$ の下界を見つけてください。

---

## 問題5：面積評価と不等式

$f(x) = \frac{1}{1+x^2}$ に対して、以下を証明してください。

$$\frac{\pi}{5} < \int_0^1 \frac{1}{1+x^2} dx < \frac{\pi}{4}$$

**ヒント**：$\frac{1}{1+x^2}$ を上下から評価し、逆正接関数の値を利用してください。

---

## 問題6：分割による面積評価

区間 $[0, 1]$ を $n$ 等分し、リーマン和を用いて以下を証明してください。

$$\int_0^1 x^2 \, dx = \frac{1}{3}$$

**ヒント**：左側リーマン和と右側リーマン和で上下から挟み込んでください。

---

## 問題7：複合関数の評価

以下の不等式を証明してください。

$$\int_0^{\pi/4} \tan x \, dx < \frac{\pi}{8}$$

**ヒント**：$\tan x < x$ などの不等式と、$x \in [0, \pi/4]$ での関数の性質を利用してください。

## 問題8：積分と面積の関係

$f(x) = e^{-x^2}$ に対して、以下を証明してください。

$$0.74 < \int_0^1 e^{-x^2} dx < 0.76$$

**ヒント**：$e^{-x^2}$ を区間ごとに上下から評価してください。

## 問題9：凸性を用いた評価

凸関数 $f(x) = x^3$ について、区間 $[0, 1]$ で以下を証明してください。

$$\int_0^1 x^3 \, dx < \frac{1}{3}$$

**ヒント**：台形公式や直線による上界を利用してください。

---

## 問題10：京大型・複合評価問題

次の不等式を証明してください。

$$\frac{e^2 - 1}{2e} < \int_0^1 xe^x \, dx < \frac{e-1}{e}$$

**ヒント**：
- 下からの評価：$xe^x$ を直線で下から押さえる
- 上からの評価：$x \leq 1$ のとき $xe^x$ を上から押さえる
- 部分積分の値を参考にしても構いません

## 別紙：解答への道筋

各問題を解く際は以下のプロセスを意識してください：

1. **関数の性質把握**：定義域での関数値の範囲、増減、凸凹を確認
2. **適切な比較関数の選定**：直線、べき関数、指数・三角関数など
3. **不等号の成立確認**：微分やグラフで両関数の大小関係を証明
4. **積分の計算**：定積分を明示的に計算して不等式を完成させる
5. **検証**：必要に応じて数値で答えを確認
